# OtiumX – AI Voice Journal & Strategic Life Insights

OtiumX is an AI-powered journaling platform that converts spoken daily reflections into structured life insights.

Features:
- Voice journaling (speech → text)
- Multi-language input (English, Bengali, Hindi, Kannada, Malayalam)
- AI-powered life pattern analysis
- Weekly strategic insight reports
- Firebase backend with secure cloud functions

Tech Stack:
- React + TypeScript
- Redux Toolkit
- Firebase Auth
- Firestore
- Firebase Functions
- Google Gemini AI
